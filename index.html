Plug.DJ Reloaded

The most feature-rich and versatile extension for Plug.DJ to date.

Download Download View Plug.DJ Reloaded on GitHub

Plug.DJ Reloaded is a Google Chrome extension (Firefox version coming soon!) that provides a whole new set of features, to make your experience on Plug.DJ as enjoyable as possible. Plug.DJ Reloaded should suit anyone's needs with features such as private messaging, a wide range of chat notifications, various functions such as autowoot, and fully customizable style sheets.

Some of the features include:

  • Greentext
  • Autowoot
  • Private Messaging
  • Background replacement
  • Userlist
  • Custom chat theme and icons
For a full list of features you can visit the features page
